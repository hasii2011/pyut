PyUt installation

Prerequisites


Windows

You have two alternatives on Windows, download the executable file which is an installer (PyutSetupXXX.exe) or work with sources (or CVS files). First option will require nothing to be preinstalled. For the second alternative, you will have to install Python 2.21and at least wxPython 2.3.42.

Linux

For Linux (or Unix) platform, you will need to have Python 2.2 and wxPython 2.3.4 installed. For download location, see Windows prerequisites.

Installation

Windows

You will have the choice to either use the source code or the installer for the Windows platform.

To use the source code directly, please read the paragraph below named "Generic install". The installer which you can download on PyUt's web site is self-extracting : you'll simply need to execute the .exe, tell where to install PyUt and start working with it.

Linux


See "Generic install" below.

Generic install


For the generic install, download the PyUt source files (.tar.gz) and decompress them using : tar xvzf distrib.tar.gz. You can use WinZip (or Cygwin) for Windows platform.

CVS


You can download the current development sources with the two following steps :
  • cvs -d :pserver:anonymous@cvs.pyut.sf.net:/cvsroot/pyut login
  • cvs -z3 -d :pserver:anonymous@cvs.pyut.sf.net :/cvsroot/pyut co pyut

To run PyUt, go in the directory $PYUTPATH/src 3 and type python pyut.pyw in a console or double-click on it in Windows.
